# 更新履歴

## 1.3.0-beta.1

* βリリースです。作者によるテストがまだ済んでいません。問題があれば報告してください。
* `wysiwyg`型（`wp_editor()`によるリッチエディタ）を追加。保存時は`wp_kses_post`でサニタイズされます。リピーターの`sub_fields`内では使用できません（`textarea`として扱われます）。

## 1.2.0

* URL、メール、日付、カラー、リピーターのフィールド型を追加
* 登録したフィールドの出力コード生成機能を追加
* フィールドのサニタイズ・バリデーションを改善
